# 妹抖酱

一个简单的AI程序。

## 关于
这个AI程序可以在系统托盘里弹出气球，用来提醒，或者只是简单的打招呼等等。

名字来源于樱花庄的宠物女孩里赤坂龙之介的妹抖酱程序。

功能上跟 SAO Project 里的 Yui 应该差不多，但是 Yui 说的话是固定的，并不能自己修改。我打算加以改进。

用户可以在 config 文件里设置自己的短语。目前先做一个中文版的试用好了，反正你也可以自己改的不是么。
